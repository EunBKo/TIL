# 꿀팁 대방츌😍

## Conemu 설치

cmd창을 이용해서 프로젝트를 진행하여도 무방하나, 이 cmd 애플리케이션을 사용하면 시작폴더위치나 복사기능 등 여러 기능을 사용할 수 있어서 편리하다.

다운로드는 공식 블로그에서 가능하다.

[ConEmu - Handy Windows Terminal](https://conemu.github.io/)
